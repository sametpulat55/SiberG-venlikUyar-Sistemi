- Öne Çıkan Özellikler
Gerçek Zamanlı Analiz: Gelen paket trafiğini anlık olarak tarar.

Saldırı Tanımlama: Şüpheli etkinlikleri (yüksek frekanslı istekler, geçersiz port denemeleri) tanımlar.

Loglama Sistemi: Tespit edilen saldırıları detaylı olarak (IP adresi, zaman damgası, saldırı türü) kayıt altına alır.

Uyarı Mekanizması: Kritik seviyedeki tehditleri önceliklendirir.

- Teknik Altyapı
Dil: Python / C# (Projeni hangi dilde yazdıysan onu seçebilirsin).

Kütüphaneler: Scapy, Socket veya WinPcap (Ağ trafiği dinleme için hangisini kullandıysan).

Analiz Mantığı: Kural tabanlı tespit algoritmaları.
- Kurulum ve Kullanım
Gereken bağımlılıkları yükleyin: pip install -r requirements.txt

Uygulamayı yönetici haklarıyla çalıştırın (Ağ trafiğini dinleyebilmek için gereklidir).

python main.py komutu ile izlemeyi başlatın.
- Güvenlik Notu
Bu araç eğitim ve ağ güvenliği farkındalığı amacıyla geliştirilmiştir. Sadece yetkiniz olan ağlar üzerinde test edilmelidir.
